# What Did You Mean? - Sound Assets Directory

Place your sound effects in this directory with the following filenames to enable them in-game:

- tap.mp3       -> General button click / tap (Soft pop)
- ding.mp3      -> Vote submitted / Success (Tiny ding)
- whoosh.mp3    -> Phase switch (Swoosh)
- beep.mp3      -> Player joined / Ready (Cheerful beep)
- buzz.mp3      -> Wrong guess / Error (Goofy buzz)
- sparkle.mp3   -> Good guess / Interpretation (Sparkle ding)
- drumroll.mp3  -> Reveal start (Drumroll)
- pop.mp3       -> Reveal moment / Item appear (Funny pop)
- tick.mp3      -> Timer ticking (Subtle clock tick)
- coin.mp3      -> Points gained (Coin sound)
- cheer.mp3     -> Winner reveal (Crowd cheer)
- confetti.mp3  -> Confetti effect (Paper pop)

Recommended Format: MP3 (compressed for web)
Style: Cartoon / Playful / Soft
